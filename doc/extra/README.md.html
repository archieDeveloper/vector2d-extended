vector2d-extended

Installation

With npm do

npm install vector2d-extended

Simple example

Addition of two vectors:

var vec1, vec2, sumVec1AndVec2;

vec1 = Vector2d(10,0);
vec2 = Vector2d(25,30);
sumVec1AndVec2 = vec1
    .clone()
    .add(vec2);

Or set the length of the vector:

var vec1, vec2, newLength;

newLength = 30;

vec1 = Vector2d(15, 35);
vec2 = vec1
    .clone()
    .normalize()
    .multiply(newLength);
